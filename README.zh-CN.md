# 简介

VUtil 是可视化语法库[VGrammar](https://github.com/VisActor/VGrammar) 的通用可视化工具。 包括数据处理、数学工具、js 代码工具、图形处理工具。

# 仓库简介

本仓库包含如下 package

1. vutil：可视化工具包合集
2. vdataset：数据处理工具
3. vvisutil：数学和图形工具

# 使用

## 安装

[npm package](https://www.npmjs.com/package/@visactor/vutil)

```bash
// npm
npm install @visactor/vutils

// yarn
yarn add @visactor/vutils
```
